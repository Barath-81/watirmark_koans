Model::Factory

The Watirmark Model Factory allows you to create new Models. You can define models to have default keyword values and use them throughout your tests.

    class SingleDonorModel < Watirmark::Model.factory
        keywords SingleDonorModelView.keywords
        traits :donor_address
        model_type Donor
        defaults do
            first_name {"John"}
            last_name  {"Smith"}
        end
    end
